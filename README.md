#LessNvim

Introduction

Can you configure Neovim with only five plugins? The answer is yes!
This idea came to me before bed. To achieve a complete Neovim environment, I deliberated over the plugins for a long time (tokyonight.nvim or git.nvim?). In the end, I gave up on git.nvim (it’s not like I can’t type commands manually) and chose a better theme.

If possible, could you follow my Bilibili account (please)? My name is NaNo-MooN, a middle school student who loves programming, art, and Neovim.
By the way, only a small part of this code was written by me. Some of it was copied from GitHub, and another part was written by AI (e.g., the file structure).

I hope you can use this configuration as a foundation to develop your own LessNvim! I also hope you can learn some new plugin and configuration methods through this setup.

Lastly, this configuration is not very user-friendly (there are simply too few plugins). Also, I won’t be actively updating this project. However, it’s a good starting point. You can add more plugins based on your needs or use more mainstream distributions (like Lazyvim, Nvchad, etc.).

Plugin List

1. nvim-tree - File tree plugin
2. telescope - Search plugin
3. tokyonight.nvim - Theme plugin
4. mason (and mason-lspconfig) - LSP management plugin
5. blink - Code snippet plugin
