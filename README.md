#LessNvim

##Introduction
Can you configure Neovim with just five plugins? The answer is absolutely yes!  
I thought of this before going to bed at night. To create a complete Neovim environment, I carefully considered the plugins for a long time (should I use tokyonight.nvim or git.nvim?). In the end, I abandoned git.nvim (it's not like I can't type commands manually) and chose a better theme.  
If possible, could you follow my Bilibili account (please), my name is NaNo-MooN, a middle school student who loves programming, art, and Neovim.  
By the way, only a small part of this code was written by me. Some parts were copied from GitHub, and other parts were written by AI (such as the file structure).  
I hope you can use this configuration as a foundation to develop your own LessNvim! I also hope you can learn some new plugins and configuration methods through this configuration.  
One last thing, this configuration actually has a poor user experience (too few plugins). Also, I won't actively update this project. However, it's a great starting point. You can add more plugins according to your needs, or use more mainstream distributions (like Lazyvim, Nvchad, etc.).

##Plugin List
1. nvim-tree - File tree plugin
2. telescope - Search plugin
3. tokyonight.nvim - Theme plugin
4. mason (and mason-lspconfig) - LSP management plugin
5. blink - Code snippet plugin
